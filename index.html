About me:

If you're wondering about who I am, I am a Statistics student that will be graduating in Spring 2021. Through specfic statistics classes and personal projects, I hope to become proficient at analysis, and eventually becoming a data analyst.

The goal of this website:

I see two major goals with this website:
  • To showcase analysis skills that I've learned
  • To mix personal interests with analysis to provide intriguing results
For the first point, this website serves as an online portfolio.
For the second point, I like the idea being able to apply concepts to things I've learned either from class or from online to hobbies I enjoy. For example, what if I wanted to see if the speed of a tennis ball serve is related to the temperature on the moon? I'm not sure if anyone on Earth would be thinking of a question like that, but it's interesting to see if there is indeed relationships between things that people don't tend to compare. If anything, we may discover a new trend that could help the world.

How this website works:

I plan on having links to projects I'll be working on. Also, I'll have some links for readings I like. Eventually, I would like to have this website turn into my main page so I don't have to keep typing in websites I use all the time in the search bar.
